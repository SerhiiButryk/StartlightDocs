## Lambda vs anonimus Kotlin classes

Did you know lambda and anonimus classes have different runtime cost?

Let's dive deeper and find out why it happens. 

# Anonimus classes:

Let's create a 'Test.kt' file with the source code:

```kotlin
fun interface Listener {
    fun call()
    fun message() = "Hello"
}

class A {
    private var listener: Listener? = null

    fun addListener(listener: Listener) {
        this.listener = listener
    }

    fun call() {
        listener?.message()
    }
}

fun main() {

    val a = A()

    // Creates an anonimus class
    a.addListener(object : Listener {
        override fun call() {
            println("Hello this is my message")
        }
    })

    a.call()

}
```

Let's check what compile will generate.

First of all we can see that a new 'TestKt$main$1.class' class file gets created.
